Co-ordinate Reference Systems

Created Wednesday 22 September 2021
@GEOGRAPHY @YEAR3 @MAP_PROJECTIONS @GEOINFORMATICS

@coordinates are a set of geometirc points.
A @coordinate_reference_system describes how the coordinates are assigned to elements in the @gis

The coordinates are numbered in a logical way such that they are unambiguous.
If the numbers are organised and meaningful then the distance and direction between them can be calculated

There are many different CRSs and the same point will have different coordinates in different systems.

These all point to the same place

  • Degrees/ Minutes/ Seconds
  • Decimal
  • Eastings/ Northings
  • Radians


Longitude and latitude are not based on a @cartesian plane (which is a flat X/Y grid) but based on a spherical coordinate system from Earth's centre.
The axes are sometimes reversed
